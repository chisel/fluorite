REST API Example

Add a new pet

This endpoint adds a new pet in the database.

Path

post /pets/add/

Authentication

Authentication is done using a bearer token obtained from the /auth/getToken endpoint.

URL Parameters

Name Type Description
database string The database name to add the pet in

Query Parameters

Name Type Description
xml boolean If present, the response type would be XML instead of JSON

Body

  • JSON
  • XML
  • Multipart
  • Form Data
  • Text
{
  "name": "The pet name",
  "type": "The pet type (dog, bird, etc.)",
  "breed": "The pet breed (optional)"
}
<Data>
  <Property1>string</Property1>
  <Property2>number</Property2>
</Data>
Key Type Required Description
name string true The pet name
type string true The pet type (dog, bird, etc.)
breed string false The pet breed
Key Type Required Description
name string true The pet name
type string true The pet type (dog, bird, etc.)
breed string false The pet breed

A CSV file with columns 'name', 'type', and 'breed'
